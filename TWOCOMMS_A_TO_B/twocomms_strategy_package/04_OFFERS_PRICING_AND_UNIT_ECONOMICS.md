# 04 — Офферы, прайс и юнит-экономика TwoComms

## 1. Цель документа

Собрать рабочую коммерческую модель TwoComms после уточнения DTF-экономики:

- DTF-плёнка;
- кастом на одежде;
- малые тиражи;
- готовая одежда;
- прайс;
- вклад;
- правила, которые защищают владельца от выгорания и демпинга.

Все цифры ниже нужно считать рабочей моделью, а не финальной бухгалтерией. Следующий агент должен проверить их, уточнить и собрать финальную финансовую модель.

## 3. DTF-плёнка: прайс

### Рекомендованный прайс TwoComms

| Объём | Цена | Вклад при COGS 165 ₴/м |
|---:|---:|---:|
| 1–4 м | 350 ₴/м | 185 ₴/м |
| 5–19 м | 330 ₴/м | 165 ₴/м |
| 20–49 м | 320 ₴/м | 155 ₴/м |
| 50–99 м | 305 ₴/м | 140 ₴/м |
| 100+ м | 295 ₴/м+ индивидуально | 130 ₴/м+ |

### Почему не использовать сетку 350 / 320 / 290 / 260 / 230

Некоторые агенты предложили опускаться до **260 ₴/м** и **230 ₴/м** на объёмах.

Это опасно:

| Цена | Вклад при COGS 165 | Риск |
|---:|---:|---|
| 260 ₴/м | 95 ₴/м | мало для соло-бизнеса с препрессом и рекламациями |
| 230 ₴/м | 65 ₴/м | почти загрузка ради загрузки |

Такой прайс может иметь смысл только для крупной фабрики, высокой автоматизации и стабильных файлов. Для TwoComms на старте — нет.

## 4. DTF-плёнка: surcharge-логика

### Базовый тариф

Базовый тариф применяется, если:

- файл готовый;
- заливка умеренная;
- нет сплошного белого ковра;
- нет срочности;
- нет сложной ручной подготовки;
- клиент соблюдает техтребования.

### Надбавки

| Надбавка | Условие |
|---:|---|
| +20 ₴/м | повышенная заливка, много элементов, много белого, но не экстремально |
| +40 ₴/м | плотные белые/цветовые блоки, большие плашки, высокая нагрузка на чернила |
| +60 ₴/м | сверхплотная заливка, почти сплошной фон, срочность, риск 8-pass/photo mode |
| +30% | день-в-день / сверхсрочно, если физически возможно |

Правило:

> Если макет выглядит как “несколько принтов на метре” — базовый тариф. Если как “печатный ковёр” — surcharge.

## 5. DTF-плёнка: препресс, правки, минимальный чек

| Услуга | Цена |
|---|---:|
| Проверка файла | бесплатно |
| 1 мелкая правка | бесплатно |
| Лёгкая подготовка файла | 100 ₴ |
| Nesting / сборка рулона / адаптация | 150–250 ₴ |
| Сложная доработка | 300 ₴/час, минимум 150 ₴ |
| Дизайн с нуля | от 500 ₴ или по отдельному расчёту |
| Повторные правки после согласования | +100 ₴/итерация |
| Минимальный B2B-чек | 500 ₴ рекомендовано |
| Минимальный технический заказ | 1 пог.м или сборный запуск |

Важное правило:

> Чужой плохой файл не должен бесплатно съедать время владельца. Если клиент не готов платить за подготовку файла, заказ лучше не брать.

## 6. DTF-плёнка: условия оплаты и SLA

| Условие | Правило |
|---|---|
| Новый клиент | 100% предоплата |
| Срочный заказ | 100% предоплата |
| Тяжёлый макет | 100% предоплата после согласования surcharge |
| Постоянный клиент | 50/50 возможно только после 2–3 успешных заказов |
| До 20 м | 1 рабочий день после оплаты и файла, если очередь позволяет |
| 20–49 м | 1–2 рабочих дня |
| 50+ м | 2–4 рабочих дня |
| Same-day | только до 12:00/15:00, маленький объём, готовый файл |

---

## 7. Заказы, которые лучше не брать

1. Скриншоты из телефона вместо нормального файла.
2. Полупрозрачные дым/градиенты без понимания DTF-ограничений.
3. Линии тоньше технологического минимума.
4. Pantone-critical заказ без теста.
5. “Нужно сегодня”, но файл ещё не готов.
6. Клиент спорит с предоплатой на первом заказе.
7. Клиент требует базовый тариф за сплошные заливки.
8. Микрозакказ ниже чека, ради которого нужно запускать линию.
9. Розничное нанесение на неизвестную одежду клиента, если нет компенсации риска.

## 11. Готовая одежда TwoComms

### Роль

Не главный cold-paid offer. Роль:

- proof;
- стиль;
- trust;
- upsell;
- контент;
- брендовая идентичность.

### SKU-ядро

Не расширять каталог. Держать:

- 4–6 футболок hero-SKU;
- 1–2 худи;
- 1 лонгслив опционально;
- black как база;
- coyote/olive/graphite как вторичные;
- один white/milk тест, если экономика и спрос подтверждаются.

## 13. Сценарии для цели 50k чистыми

Допустим, нужно покрыть условно 60k валового вклада в месяц до личных/операционных целей. Точная сумма зависит от fixed costs и грантового периода.

### Только DTF-плёнка

При вкладе 165 ₴/м:

```text
60000 / 165 = 364 м/мес
```

Это около:

```text
364 / 22 рабочих дня = 16,5 м/день
```

Реально технически возможно, но коммерчески нужно много B2B спроса.

### Только кастомные футболки

При среднем вкладе 650 ₴:

```text
60000 / 650 = 92 заказа/мес
```

Это около:

```text
92 / 30 = 3,1 заказа/день
```

Возможно по спросу, но тяжело по коммуникации, если всё вручную.

### Только худи

При среднем вкладе 1200 ₴:

```text
60000 / 1200 = 50 худи/мес
```

Сезонно возможно осенью/зимой, но не как единственная модель.

### Гибридная модель

Пример:

| Направление | Вклад |
|---|---:|
| DTF: 150 м × 155 ₴ | 23 250 ₴ |
| 40 кастом-футболок × 650 ₴ | 26 000 ₴ |
| 10 худи × 1200 ₴ | 12 000 ₴ |
| **Итого** | **61 250 ₴** |

Это выглядит реалистичнее для соло-основателя, чем одно направление.

## 14. Финальный коммерческий вывод

1. **DTF-плёнка маржинальна, если не демпинговать и не делать бесплатный препресс.**
2. **Кастом даёт лучший front-offer и быстрый cash.**
3. **Готовая одежда должна продавать доверие и стиль, а не быть главным performance-направлением.**
4. **Гибрид даёт лучший путь к 50k+ чистыми, потому что распределяет нагрузку между метрами, заказами и брендовым proof.**
