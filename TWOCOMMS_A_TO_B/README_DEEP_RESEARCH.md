## 2. Бизнес за 60 секунд
- **Бренд:** TwoComms, Харьков, Украина. Стритвир (футболки, худи, лонгсливы) со смысловыми принтами.
- **ДНК:** характер / мотивация / «продолжение после критической точки» («две запятые» в названии).
  Основатель — боевой ветеран (бой с 2022, срочка 2015); ветеранство = слой доверия и PR, НЕ главный
  лозунг (бренд продаёт мотивацию и стиль, а не «военку»).
- **Производство:** собственный DTF-принтер + термопресс (с ~сентября — авто-пресс + горячая плёнка).
  Планируется вышивальная машина (грант, ~через 1.5 мес) и женская линия.
- **Два разных продукта-направления (НЕ путать):**
  - **B2C-бренд + кастом-принт на одежде** (основной сайт). Кастом уже даёт продажи органикой.
  - **B2B-плёнка DTF** (субдомен) — продажа готовой плёнки бизнесам, которые сами прессуют. Пока 0 продаж.
- **Инфраструктура (важно):** Django-монолит с 4 поддоменами — магазин, DTF, CRM (~90 моделей,
  телефония с AI, 295k спарсенных лидов), финкабинет, склад. **Инфраструктура переразвита относительно
  спроса.**
- **Деньги:** грант оплачивает аренду офиса до декабря 2026; на счетах ~496к ₴ (грант-остаток + пенсия);
  ожидается выплата за ранение ~300к. **Капитал — не главный ограничитель.**
- **Главный диагноз:** бизнес ограничен **спросом и фокусом основателя**, не деньгами и не мощностью.

## 4. Ключевые факты и цифры (опорная таблица)

> **Все цифры — для проверки и при необходимости оспаривания внешними данными.**

### Спрос и продажи
- 🟢 ~5 заказов/мес зафиксировано в системе (39 заказов за ~8 мес); ~200-300 за всю жизнь бренда.
- 🟢 Валидированная выручка: 34 выполненных заказа = 53 785 ₴; **AOV ≈ 1582 ₴**; ~1.26 позиций/заказ.
- 🟢 **0 заказов с платной рекламы** (utm пуст во всех заказах). Топ-продажи — разовый коллаб с воен.частью.
- 🟢 Возвраты <1%. Логистика (Новая Почта) — НЕ узкое место.
- 🟢 Кастом-принт на одежде: **≥3 заказа по ~1100 ₴** органикой (AI/ChatGPT-трафик) + 2 в переписке.
- 🟢 DTF-плёнка B2B: **0 заказов** (примерочная-конструктор: 67 сессий → 0 заказов плёнки).
- 🟢 Отзывов на сайте: **0** (хотя технически витрина отзывов готова).

### Экономика (реальная себестоимость — НИЖЕ, чем в старом аудите)
- 🟢 Бланк футболки: ~300 ₴ (база) / 500-510 (классика-резинка) / ~300 (CRC). Бланк худи: ~950-1000.
- 🟢 Печать на изделие (сырьё): ~15-30 ₴ (НЕ 90 — то была цена услуги). Упаковка ~26. Брак футболок 2-4%.
- 🟢 Цены (решения владельца): футболка **1000 ₴**, худи **2000 ₴**, кастом ≥1000 (≥1150 «с нуля»),
  плёнка B2B **310-350 ₴/пог.метр**.
- 🟢 Вклад с единицы: футболка@1000 **~633 ₴** (маржа 63%), худи@2000 **~912** (46%), плёнка@330/м
  **~255** (маржа 77%).
- 🟢 **Амортизация (правка 20.06, файл 02 §6b):** принтер ~600к — НЕ вшивать в цену единицы (при 20 ед/мес
  это 625 ₴/ед, при 450 ед/мес — 28 ₴/ед; обвально зависит от объёма). Трактовать как период-расход
  (~12,5к/мес) и капитал к окупаемости. «Убыточен» не товар, а недозагрузка станка → решает ОБЪЁМ, не цена.
- 🟢 **Скрытый расход — печатающая головка** (XP600/i3200): ресурс ~4-6 мес, замена $300-500 → ~3к/мес.
  Ранее не учитывался. Электричество на принт — ничтожно (~0,5-1 ₴); значимое электричество = overhead помещения.
- 🟢 **Реклама (файл 08):** старт оптимизировать на СООБЩЕНИЯ; при конверсии ~15% `CAC = цена_сообщения/0.15`;
  держать цену сообщения < $1.5 (зелёный), > $3 — стоп. Переход на «Покупки» при 30-50 покупках/нед.

### Деньги и постоянные

## 5. Стратегия в одном экране (что предложено — проверь и улучши)
- **Принцип:** «перестать строить — начать загружать». Спрос — единственная настоящая задача.
- **3 ноги дохода:** (A) платная **видеореклама B2C**, старт с футболок (сезон), непрерывно 8-12 нед,
  ведёт на комплект/порог бесплатной доставки от 2000; (B) **плёнка B2B** через лид-лендинг + аутрич,
  ставка на постоянных клиентов (LTV, маржа 77%); (C) **консигнация** тест-партиями (объём без рекламы).
- **AI/SEO** — дешёвый усилитель (ChatGPT уже ~11% трафика, кастом-заказы это подтверждают).
- **Окно гранта:** заработать + накопить буфер на январь + построить двигатель спроса ДО декабря.
- **Апсайд (не база плана):** вышивка → якорный клиент + премиум-B2B; женская линия; кепки.
- **Фазы:** Фундамент (сейчас→июль) → Спрос (август→сентябрь) → Масштаб (сентябрь→декабрь).
- **Критично для исполнения:** план учитывает прокрастинацию владельца (ритуалы, делегирование
  операционки, бюджет-кап на стройку инструментов — см. файл 06).
